Após confirmação do pagamento, você recebe por email: instalador + chave de ativação + guia de instalação. Tudo em até 1 hora útil. Também pode receber via WhatsApp se preferir.
Os softwares funcionam sem internet? +
Sim! As versões Offline funcionam 100% sem internet — seus dados ficam salvos no próprio computador. As versões Online precisam de internet para sincronizar dados entre vários computadores da equipe.
Qual a diferença entre versão Offline e Online? +
Offline: instalado no seu computador, funciona sem internet, dados ficam só no seu PC. Ideal para uso individual.

Online: acesso via navegador em qualquer lugar, banco de dados na nuvem, permite vários usuários (toda a equipe), backup automático. Ideal para equipes.
